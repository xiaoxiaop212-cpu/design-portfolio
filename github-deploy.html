GitHub Pages 部署指南

1
创建 GitHub 仓库
访问 GitHub，创建一个新仓库，命名为 design-portfolio 创建仓库
2
推送代码到 GitHub
在项目文件夹中运行以下命令： git remote add origin https://github.com/你的用户名/design-portfolio.git git branch -M main git push -u origin main
3
启用 GitHub Pages
进入仓库 Settings → Pages → Source，选择 "GitHub Actions"
4
等待部署完成
进入仓库 Actions 标签页，等待部署完成（约 2-3 分钟）

✅ 部署完成后，你的网址将是：

https://你的用户名.github.io/design-portfolio/

💡 提示：部署配置已自动完成，推送代码后会自动触发部署

部署完成后，将获得的网址告诉我，我可以帮你生成二维码！
